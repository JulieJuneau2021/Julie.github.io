HTML Notes

1. Header - h1, h2, h3
In the header tag is where you write the header of your page
2. Body - body
In the body tag is where you write the body of your paper or article
3. Paragraph - p
In the p tag is where you would write a paragraph
4. Ordered List - ol
In the ol tag you write an ordered list of items to be displayed in order also using the li tag
5. Unordered List - ul
In the ul tag is Where you can display a list of items with no order also using the li tag
